# 淮滨县淮河生态农业发展投资有限公司网站

## 项目简介
本项目是淮滨县淮河生态农业发展投资有限公司的官方网站，使用Vue.js开发。

## 功能特点
- 公司信息展示
- 招标采购信息发布
- 企业动态新闻
- 投资项目展示
- 联系方式

## 技术栈
- Vue 3
- Vue Router
- Vite

## 本地开发
```
cd web
npm install
npm run dev
```

## 构建部署
```
cd web
npm run build
```

## 目录结构
- web/ - 前端Vue项目
  - src/ - 源代码
    - components/ - 组件
    - assets/ - 静态资源
    - router/ - 路由配置
